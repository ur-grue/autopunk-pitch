# 4-Pass Revision Framework

## Pass 1 — Developmental (Big Picture)

Focus: Does the story WORK?

Checklist:
- [ ] Protagonist arc: clear want/need, tested at midpoint, resolved at climax
- [ ] Stakes escalation: each act raises the stakes
- [ ] Pacing: no dead zones (3+ consecutive low-tension chapters)
- [ ] Subplot resolution: every thread addressed
- [ ] Plot holes: no unresolved setups or contradictions
- [ ] Theme: present but not preachy

## Pass 2 — Scene-Level

Focus: Does every scene EARN its place?

Checklist:
- [ ] Scene purpose: every scene advances plot OR deepens character (ideally both)
- [ ] POV consistency: no unintentional head-hopping
- [ ] Conflict: every scene has tension (even quiet scenes have internal conflict)
- [ ] Entry/exit: enter scenes late, leave early
- [ ] Scene-sequel structure: Goal -> Conflict -> Disaster -> Reaction -> Decision

## Pass 3 — Line-Level

Focus: Is the PROSE sharp?

Targets:
- Filter words: "seemed", "felt", "noticed", "realized", "began to", "started to"
- Passive voice: "was written" -> "wrote", "was seen" -> "saw"
- Adverb pruning: cut adverbs that tell what the verb should show
- Show/tell: "She was angry" -> show anger through behavior
- Repetition: same word within 3 sentences, same metaphor within a chapter
- Cliches: any phrase that could appear in a different novel without anyone noticing

## Pass 4 — Polish

Focus: Does it SOUND right?

Checklist:
- [ ] Read aloud (mentally): awkward rhythms, tongue-twisters, monotonous patterns
- [ ] Sentence variety: mix short punchy with longer flowing
- [ ] Paragraph rhythm: vary paragraph length for visual texture
- [ ] Dialogue distinction: can you tell who is speaking without tags?
- [ ] Opening hook: does chapter 1, paragraph 1 grab attention?
- [ ] Closing resonance: does the final paragraph land with weight?

## The 9 Rewriting Technique Codes

Use these codes in revision notes for precision:

| Code | Technique | Description |
|------|-----------|-------------|
| CUT  | Cut       | Remove entirely — adds nothing |
| COMP | Compress  | Say the same thing in fewer words |
| EXPAND | Expand  | Needs more detail or development |
| SHOW | Show      | Replace telling with showing |
| SPEC | Specify   | Replace vague with concrete |
| VOICE | Voice    | Adjust to match character's voice |
| PACE | Pace      | Speed up or slow down |
| MOVE | Move      | Relocate to better position |
| MERGE | Merge    | Combine with another element |
