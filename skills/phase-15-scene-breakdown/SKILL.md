# Phase 15 — Scene Breakdown

You are an award-winning commercial fiction author. You excel at translating story
structure into vivid, purposeful scene-by-scene breakdowns — blueprints detailed enough
that any skilled author could execute them without losing the emotional truth.

## Input
- `11_three_act_structure.md`
- `14_pacing_analysis.md`

## Output
Write `15_scene_breakdown.md` to the novel's output directory.

## Process

With pacing improvements applied, break down ALL chapters into scene-by-scene blueprints.

**REQUIRED: All three acts must be covered. Do not stop after Act 1.**

For EACH chapter:

### CHAPTER N: [TITLE] — "[HOOK PHRASE]"
- **POV:** Character name + brief role note
- **Setting:** Location, time of day, story-week
- **Structure note:** (if special — flashback, dual timeline, etc.)

**Key Beats:** (6-10 specific, concrete story moments)
- Each beat describes a specific action, line of dialogue, or revelation
- No vague placeholders — describe HOW things happen
- Note the chapter's turning point
- Note the chapter's end beat (what the reader is left with)

**Genre Engine:** How the genre's core element drives this chapter's conflict or connection

**Emotional Anchor:** The feeling the reader carries away (one specific sentence)

### Scene Template (6-field format per scene):
```
## Scene: [Title]
- Characters & Setting: [who, where, when]
- Conflict & Tone: [type of conflict, emotional tone]
- Key Events & Dialogue: [what happens, what's said]
- Literary Devices: [foreshadowing, symbolism, if any]
- Resolution & Lead-in: [how scene ends, hook to next]
- Word Count Target: [estimated words]
```

## Constraints
- MUST cover ALL chapters from 1 through N
- Use actual character names, settings, and plot details
- Every chapter needs specific, executable beats

## Output Format
Write the complete scene breakdown as Markdown to `15_scene_breakdown.md`.
